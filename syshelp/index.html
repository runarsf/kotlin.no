The reason for this is because strings; let's say you're trying to test for this expression: ``\[.*\]``. This would end up as ``.*``, as the brackets were escaped in the string, hence excluding them from the regex. In order to fix this, you have to escape the escape characters themselves: ``\\[.*\\]``.
